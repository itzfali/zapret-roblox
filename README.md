# Установка вручную
Если вышла новая версия Zapret, а тут нету её

Вам понадобится оригинальный Zapret
1. Скачайте [архив](https://github.com/itzfali/zapret-roblox/archive/refs/heads/main.zip) и распакуйте его
2. Откройте папку zapretrobloxlists
3. Скопируйте ВСЁ содержимое в папку lists в оригинальном Zapret (если файлы уже существуют, замените, так и задумывалось)
4. Пробуйте разные .bat файлы, пока у вас не заработает Roblox
5. Когда он заработал, откройте service.bat и включите Game Filter, и установите тот .bat файл, который сработал
